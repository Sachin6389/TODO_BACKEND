# 🚀 Backend API (Node.js + Express + MongoDB)

This is a REST API backend built using **Node.js, Express, and MongoDB** with authentication and middleware support.

---

## 📦 Features

- User Authentication (JWT)
- Protected Routes
- CRUD APIs
- MongoDB Database Integration
- Middleware-based security
- RESTful architecture

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- dotenv
- CORS
